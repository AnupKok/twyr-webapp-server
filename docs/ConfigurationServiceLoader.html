ConfigurationServiceLoader

The specialized loader for the Twyr Web Application Server Configuration Service.

Constructor

new ConfigurationServiceLoader()

Runs load / init / start in one shot so that the rest of the codebase can access module-specific configurations by the time they are loaded by the framework.
